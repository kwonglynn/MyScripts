RDKit

RDKit is a open source toolkit for cheminformatics. It can generate descriptorfor machine learning.

References:
  1. Getting Started with the RDKit in Python
  2. RDKit Cookbook
  3. RDKit Book
Examples:
1. Create an environment for the RDKit package from the rdkit channel:

conda create -n rdkit -c rdkit rdkit

2. Activate the rdkit environment:

conda activate rdkit

3. Read a single molecule:

>>> from rdkit import Chem
>>> m = Chem.MolFromSmiles('Cc1ccccc1')
>>> m = Chem.MolFromMolFile('data/input.mol')

4. Read a set of molecules, for instance, from a SDF file:

>>> mols = Chem.SDMolSupplier('data/5ht3ligs.sdf')
>>> for mol in mols:
      if mol is None: continue
        print(mol.GetNumAtoms())

5. Set the name of a molecule:

>>> m = Chem.MolFromSmiles('Cc1ccccc1')
>>> m.SetProp("_Name","cyclobutane")

6. To add and remove hydrogens:

>>> m = Chem.MolFromSmiles('Cc1ccccc1')
>>> m_H = Chem.AddHs(m)
>>> m_noH = Chem.RemoveHs(m_H)

7. Print the MDL Mol block:

>>> m = Chem.MolFromSmiles('Cc1ccccc1')
>>> print(Chem.MolToMolBlock(m))

8. Convert molecule to canonical SMILES:

>>> m = Chem.MolFromMolFile('data/chiral.mol')
>>> Chem.MolToSmiles(m)
  'C[C@H](O)c1ccccc1'
>>> Chem.MolToSmiles(m,isomericSmiles=False)
  'CC(O)c1ccccc1'

9. Write to a MOL/SDF file:

>>> w1 = Chem.SDWriter('molecule1.mol')
>>> w1.write(m)
>>> w1.close()
>>> w2 = Chem.SDWriter('molecule2.sdf')
>>> for m in mols:
      w2.write(m)
>>> w2.close()

Note: use &emsp;to add a horizontal tab to the code. Also use HTML Entities to display &emsp;.

10. Compute the 2D coordinates for a molecule. The resulting coordinates are stored on each atom of the molecule (in place).

>>> from rdkit.Chem import AllChem
>>> AllChem.Compute2DCoords(m)

11. Draw molecules,and save as an image file:

>>> from rdkit.Chem import Draw
>>> m = Chem.MolFromSmiles('Cc1ccccc1')
>>> Draw.MolToFile(m, 'molecule.png')

12. Generate the MACSS fingerprint:

MACCS (Molecular ACCess System) keys are 166 bit structural key descriptors in which each bit is associated with a SMARTS pattern. Literature

>>> from rdkit.Chem import MACCSkeys
>>> m1 = Chem.MolFromSmiles('C1(=C(SSC1=O)Cl)Cl')
>>> maccs = MACCSkeys.GenMACCSKeys(m1)
>>> maccs_bits = maccs.ToBitString()

PaDEL-Descriptor

A software to calculate molecular descriptors and fingerprints. The software currently calculates 1875 descriptors (1444 1D, 2D descriptors and 431 3D descriptors) and 12 types of fingerprints (total 16092 bits). The descriptors and fingerprints are calculated using the Chemistry Development Kit (CDK) with additional descriptors and fingerprints. CDK on GitHub.

Usage:

Launch the software from here (recommended but requires Java Web Start) or download it from here. If you download the zipped file, unzip the files to any directory and launch the software using:

java -jar PaDEL-Descriptor.jar

Cinfony

Cinfony presents a common API to several cheminformatics toolkits. It uses the Python programming language, and builds on top of Open Babel, RDKit, CDK, Indigo, JChem, OPSIN and cheminformatics webservices.

Note: cinfony is not actively updated on GitHub.
